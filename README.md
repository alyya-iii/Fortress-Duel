## Features
- Can be played in two modes: single player or two player.
- Multiple backgrounds for the player to choose from.
- Timed with limited number of cannonballs that can be shot at varying velocities.
- Winner is displayed by evaluating which player’s castle has more intact blocks at the end of the match.
- Ensures the application is easy to understand.

---

## How to Run
1. Install Python (if not installed)
2. Install pygame:
   pip install pygame
3. Open the project in PyCharm or VS Code
4. Run main.py
